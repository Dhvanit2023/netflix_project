NETFLIX

About This Project

This is a B.Tech AIT student project developed using FastAPI and Cloudinary.

The main objective of this project is to understand how online video streaming platforms like YouTube and Netflix work, including concepts such as video delivery, streaming, and network protocols.

This project demonstrates the basic architecture of a video streaming system, including backend API handling, video storage, and frontend playback.

Developed by Dhvanit Patel.

  • Backend: Python (FastAPI)
  • Frontend: HTML, CSS, JavaScript
  • Video Storage: Cloudinary
  • Streaming Concept: HTTP-based streaming (HLS concept)
  • Features: Video upload, thumbnail generation, video playback, basic authentication
